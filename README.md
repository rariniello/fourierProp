## Development setup

Clone the repository

```sh
git clone <repo> 
```

Navigate into the repository's directory

```sh
cd fourierProp
```

Create the virtual environment

```sh
conda env create -f requirements.yml
```

Activate the virtual environment

```sh
conda activate fourierProp
```
